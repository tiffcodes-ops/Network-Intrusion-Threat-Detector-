# Cyber Threat Detection using CICIDS2017 Dataset

This project utilizes machine learning and data visualization to detect and analyze cyber threats in network traffic, leveraging the CICIDS2017 dataset.

## Features
- Data cleaning and preprocessing
- Supervised learning (Random Forest)
- Unsupervised anomaly detection (Isolation Forest)
- Correlation heatmap and attack visualization
- Streamlit dashboard

##  Dataset
- **Source**: CICIDS2017 dataset 
- Used: Friday.csv
